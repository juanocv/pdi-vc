dist1.py TD iterativa por erosões sucessivas 
def dist1(f, b=np.ones((3,3), dtype='uint8')):
    """TD iterativa por acúmulo de erosões sucessivas.
    Métrica induzida pelo elemento estruturante b."""

    # 1. Garantir entrada binária {0, 1}
    img_corrente = (f > 0).astype('uint8')

    # 2. Acumulador de erosões suportadas
    acumulador = np.zeros_like(f, dtype='uint16')

    # 3. Erosões sucessivas até extinção do objeto
    while np.any(img_corrente):
        acumulador += img_corrente        # conta presença
        img_corrente = cv2.erode(         # descasca camada
            img_corrente, b
        )

    # 4. Retornar no menor tipo possível
    if acumulador.max() <= 255:
        return acumulador.astype('uint8')
    return acumulador
1
(f > 0).astype('uint8') — normaliza qualquer imagem binária para {0,1}; necessário pois cv2.erode opera sobre esses valores.
2
acumulador += img_corrente — em cada iteração, pixels ainda vivos somam 1. Ao fim do loop, acumulador[x,y] = quantas erosões o pixel (x,y) sobreviveu = distância ao fundo.
3
cv2.erode(img, b) — aplica erosão morfológica com elemento estruturante b. A geometria de b define a métrica: cruz → L₁ (Manhattan), quadrado → L∞ (Chebyshev).
4
O número de iterações é igual ao valor máximo da TD — proporcional ao raio do maior objeto. Para imagens grandes, use cv2.distanceTransform(f, cv2.DIST_L2, 5).
Comparação: dist1 vs. dist (OpenCV)
dist1 — iterativa
Métrica configurável via b. O(r·N) onde r = raio máximo. Didática e exata na métrica de b.
dist — OpenCV L₂
Euclidiana exata, O(N) com dois passes. Recomendada para imagens de alta resolução.
