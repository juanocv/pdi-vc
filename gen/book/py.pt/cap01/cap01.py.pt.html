1.6 O que Г© uma Imagem Digital?

Uma imagem digital Г© formada por uma grade de pixels (Picture Elements), onde cada pixel Г© a menor unidade elementar da imagem.

No computador, isso resulta em trГӘs matrizes sobrepostas (canais R, G e B), onde cada cГ©lula contГ©m um valor de intensidade para aquele canal especГӯfico no pixel.

1.10 ConversГЈo de Tipos e LimiarizaГ§ГЈo

Conforme vimos, uma imagem colorida no espaГ§o RGB Г© representada pela funГ§ГЈo:

\[ f: \mathbb{E} \to \{0,1,\dots,255\}^3 \]

Ou seja, para cada pixel \((x,y)\), temos trГӘs valores \((R,G,B)\) que definem sua cor.

Para converter uma imagem RGB em tons de cinza (grayscale), Г© necessГЎrio combinar os trГӘs canais em um Гәnico valor de intensidade \(g\), que representa o brilho percebido. ApГіs o cГЎlculo, o valor \(g\) Г© arredondado para o inteiro mais prГіximo e ajustado ao intervalo \([0, 255]\). O resultado Г© uma nova imagem, agora em tons de cinza, representada por:

\[ f_{\text{cinza}}: \mathbb{E} \to \{0,1,\dots,255\} \]

A partir da imagem em tons de cinza \(f_{\text{cinza}}(x,y)\), uma operaГ§ГЈo fundamental Г© a limiarizaГ§ГЈo, que produz uma imagem binГЎria (apenas preto e branco). Para isso, escolhe-se um valor de corte \(T\) (geralmente no intervalo \([0,255]\)) e define-se:

\[ f_{\text{bin}}(x,y) = \begin{cases} 255 & \text{se } f_{\text{cinza}}(x,y) > T \\[4pt] 0 & \text{caso contrГЎrio} \end{cases} \tag{1.4}\]

Exemplo: Com \(T = 128\), pixels com intensidade acima de 128 tornam-se brancos (255); os demais tornam-se pretos (0).

A limiarizaГ§ГЈo Г© amplamente usada para segmentar objetos do fundo, extrair bordas ou criar mГЎscaras binГЎrias para processamento posterior.

Nota: O valor 255 representa o branco mГЎximo em imagens de 8 bits, enquanto 0 representa o preto absoluto.

A biblioteca mm oferece uma alternativa automГЎtica: quando o parГўmetro limiar nГЈo Г© fornecido, a funГ§ГЈo mm.threshold(img_gray) calcula o valor de \(T\) pelo mГ©todo de Otsu (Otsu, 1979). # img_bin_otsu = mm.threshold(img_gray)

    [img_gray, img_bin_fixo, img_bin_otsu],

1.17.5.1 рҹ“Ҡ A Matriz de ConfusГЈo

Esta matriz organiza as contagens de acertos e erros do modelo, comparando a Classe Verdadeira (Realidade) com a Classe Predita (O que o modelo disse):

Predita Positiva Predita Negativa
Verdadeira Positiva VP FN
Verdadeira Negativa FP VN

Onde:

  • VP (Verdadeiros Positivos): Classificou corretamente como positivo.
  • FN (Falsos Negativos): Errou ao classificar positivo como negativo.
  • FP (Falsos Positivos): Errou ao classificar negativo como positivo.
  • VN (Verdadeiros Negativos): Classificou corretamente como negativo.

Sua tarefa:

  1. Leia 4 valores inteiros na seguinte ordem: VP, FN, FP, VN.
  2. Calcule as trГӘs mГ©tricas abaixo:

\[\text{AcurГЎcia} = \frac{VP + VN}{VP + VN + FP + FN}\]

\[\text{PrecisГЈo} = \frac{VP}{VP + FP}\]

\[\text{Sensibilidade} = \frac{VP}{VP + FN}\]

  1. Imprima os resultados formatados com duas casas decimais.

рҹ“Ң Importante:

  • A ordem de leitura das variГЎveis Г© fundamental para o cГЎlculo correto.
  • A saГӯda deve conter apenas os valores numГ©ricos, um por linha.
  • Arredondamentos devem seguir o padrГЈo da linguagem.
